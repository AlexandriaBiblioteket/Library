Alexandria library API


http://localhost:8080/alexandriabiblioteket/admin/
/newmedia POST consumes a JSON media object, add a media (book etc) to database
/listperson/{id} GET id of person, get the facts about a person
/listpersonsloan/{id} GET id of person, gets the loans of a person
/newperson/ POST consumes a JSON person object, add a person to the database
/listperson/{id} DELETE id of a person, removes person from database
/updateperson/{id}/{newname} PUT id of a person and a new name, change the name of a person
/media/{id} DELETE id of media, removes media object from database
/updatemedia/{id}/{status} POSTid of media and change of media status number, change the status of an media object

http://localhost:8080/alexandriabiblioteket/loan/
/new/ POST Requires {id} of person and {id} of media to loan. Creates a loan of a media object.
/renew/{id} PUT Requires the {id} of the media object which loan time should be extended.
/return/{id} GET id of a media object, to return to the library.
/reserv/ POST id of person to reserve the media object of a id. (Not implemented)!
